Microsoft
Microsoft has recently been well known for their involvement in all three types of cloud computing, being IaaS, PaaS, and SaaS. Microsoft is also known for their involvement in artificial intelligence, and the software part of cloud computing called "the intelligent cloud." Microsoft has been focusing more on cloud computing with Azure and Office 365, and incorporating "the intelligent web" into their cloud computing products and services.
Amazon Web Services (AWS)
Amazon is a major leader in cloud computing. It has been in this business since 2002, when it started its cloud based service, AWS. Since then, it has created the first IaaS platform and has gone even further with its shopping website. Amazon also has created multiple IaaS and SaaS services such as Elastic Compute Cloud (EC2), Elastic Beanstalk, Simple Storage Service (S3), and Elastic Block Store (EBS). Its services even extend to cloud-based artificial intelligence, called Amazon Alexa.
Salesforce
Salesforce and microsoft lead a fast growing branch of cloud computing, Saas (Software as a service). Salesforce "is a Customer Relationship Management (CRM) Platform." Salesforce specializes in cloud-based applications for sales, service, and marketing. They specialize in customer managment and satisfation. Salesforce needs no IT to run, and you can set up and go - an advantage to cloud-based software.
Google Cloud Platform
Google Cloud Platform is a cloud computing service that uses the same infrastructure as regular G-Suite services like Youtube, Gmail, and Drive. This service uses AI, machine learning, data analysis, and more. It has lots of products like the Compute Engine, that is a Iaas providing Virtual machines, BigQuery for large data analysis, and more. Google Cloud Platform is a very well rounded cloud computing service that could be compared to many other services like Microsoft Azure, and Amazon Web Services.

Jobs and Careers

Software Engineer

Software engineers are the ones that actually design and create the code required to run certain applications. They’re the ones who fix bugs or add new features. Software engineers are expected to be effective collaborators, be able to efficiently test software, and have decent analytic skills. Cloud computing is wholly dependent on the success of these engineers since they create the very platforms that cloud computing is built on. Without their ability to successfully implement the principles of computer science in the real world, our civilization would not be the same.

Data Engineer

A data engineer is the gateway to business insight. Data engineers build the data infrastructure to answer business questions. There should be entry, mid, and senior level opportunities in growing teams. They make, maintain, and construct scalable data management systems. Data engineers also make high performance algorithms and integrate the software into the data infrastructure.

System Administrator

The system administrator maintains various IT systems. This may cover system provisioning, configuration, maintenance, and decommissioning. Also, this position may be undergoing the most rapid change of the bunch.” The Systems Administrator “is the nuts and bolts” of the system, making sure the hardware and software is working and up to date. The Systems Administrator is responsible for maintaining web apps, emails, security, and more. The systems administrator is the go-to when a client has a problem or something goes wrong.
