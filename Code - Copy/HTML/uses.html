Viability

Cloud computing allows businesses to focus on their growth and success rather than focus on the complexities of maintaining and running their own IT services. This saves the business from buying additional hardware and services, allowing them to invest in more important things. Cloud computing also gives the users a platform that’s easily accessible, meaning that you don’t have to install or configure additional software. Plus, cloud computing is an important choice to consider for businesses in this day of age, since you can expand your business with the snap of a finger and improve its security as well. A study conducted by Exact, a software company, and Pb7 Research validated this by finding that small businesses who adopted cloud computing doubled their profits while seeing 25 percent of additional revenue growth. Cloud computing is extremely viable option to consider. It has survived due to its versatility and because it provides existing infrastructure and services that improve the productivity of companies. Plus, companies only have to pay for the services they use from their provider.

Major Users

Apple
Apple has been with computing since the release of iTools in 2000 and has been using it ever since. Today, Apple uses cloud computing with its communication software, Mac.com, and also has Apple pay, an app which allows you to pay people wirelessly. Also, Apple uses private clouds to store files transferred there from devices.

Google
Google is both a provider and user of cloud computing. They use their services, including google suite, for communication, payment, collaboration, and content creation. Also, Google uses hybrid clouds with its google drive application, so your work is secure but can also be shared to a public cloud (accessable by the public).

Facebook
Facebook is social networking platform that connects people across the globe. It uses cloud computing for website hosting and also to find friends, so the people using it are able to communicate effectively. Also, Facebook uses a hybrid-type cloud storage system, where you can save pictures and release them on site.

Netflix
Netflix uses cloud softwares that other companies have created, such as Amazon's AWS (which it is now leaving), and uses Apache code bundles for Netflix's open cloud service. It's primary use for it is to stream videos across the internet effectively. Netflix also uses scalability from the cloud since it realized internet traffic was too severe.

Wells Fargo
Wells fargo is a banking company with multiple franchises in the cloud. For example, Wells Fargo has made apps that allow people to pay and store their credit cards wirelessly from their phones, using the cloud. Cloud computing gives Wells Fargo the infrastructure to maintain these apps and also gives security to its users.

Twitter
Like all social media platforms, Twitter uses the cloud for a significant amount of their operations (up to 95 percent), for communication and data analysis. It uses most of the computing to analyze user profiles and comparing them to create possible connections. Twitter uses apache to analyze and interpret the data.

Disadvantages

Flexibility

The cloud has gained alot of support over the years due to many things, but most of all, because of its flexibility. This means that the files needed can be accessed anywhere, at any time, instead of at a certain place during a certain window of time.

Servers Could Go Down

When the servers go down, everything on the server is inaccessible. And since it is entirely random, a server crash could happen right before a project is due. To prevent all of the data from being lost, companies backup their data in case this happens.

Efficiency

Cloud computing is known to be very efficient. This is because the cloud services provider is responsible for the maintainance and updates, removing all the hassle for the company. Also, cloud services are usually organized, so there aren't any problems finding things.

Hacking

The cloud isn’t always secure and is thus vulnerable to hacking. To keep data from being uncovered, unique passwords along with a cloud service with a secure server should be chosen. Companies should always be vague about their data and should avoid giving specifics if not required.

Improved Collaboration

With the cloud, communication has become much easier. Now, people can communicate on websites, call others, and write emails. Because of all of this communication, more people collaborate on projects which allows for more productivity.

Loss of Data Control

When people are moving services and software to the cloud, they end up giving all of their information to the company providing the cloud service. This can be a risky move for a company looking for a cloud provider since data on the cloud isn’t always secure.

Smaller Expenses

Cloud computing is very cost efficient because it deems buying hardware and IT services unnecessary. This, in turn, lowers the cost of cloud computing and lowers the hassle as well. This has made the cloud a cheap and useful source of computing.

Unreliable

Cloud computing isn’t always the most reliable. For example, when a cloud service is unresponsive, you can’t really do anything about it. To make sure that unreliability isn’t as bad of a problem, companies should keep up with their cloud provider so they can prepare for any liabilities.
