IaaS
PaaS
SaaS

Infrastructure as a Service is the basic use of physical structures and facilities needed for the operation of a society or enterprise. Infrastructure as a Service is when you rent storage, server, operating system, etc., from a wireless provider. IaaS gives you the computing power to analyze massive amounts of data which are very useful to scientific studies or businesses looking to improve marketing strategies.

Platform as a Service is a bit broader than IaaS, since it is part of Platform as a Service. PaaS is based off of computer platforms such as google docs. It is used with operating systems, development tools on a computer, database management, business analytics, and all of the uses listed under Infrastructure as a Service. Private PaaS servers allow people to manage applications while still keeping privacy. Some companies that use private PaaS servers are Google, Amazon, and Windows.

Software as a Service is like a bank. It protects the valuables and privacy of its customers, while providing reliable service. SaaS is where something is downloaded as a software onto a device. This counts apps, launchers, and all of the uses listed in PaaS and IaaS. SaaS is also a way of transmitting applications over the cloud, hence the word software is in its name. Because of this, Software as a Service frees up time and hassle from managing software and hardware. Some advantages of this service are its painless and hassle free upgrades, its low initial cost, its seamless integration into the computer system, and that it is adopted by many.

Deployments

Hybrid Cloud
Hybrid clouds is a combination of the private cloud and public cloud and is made up of one or more public cloud providers. It allows data to move between private and public clouds, which enables more flexibility and deployment. Hybrid clouds can also manage security issues, and is known to be very efficient.
Public Cloud
Public clouds are owned and operated by a third-party provider. In this type of cloud service, the provider has complete management over the service. Though the public cloud has a reasonable amount of security, it’s not the best option if security is a concern. It's services are usually offered as either pay-per-usage or without cost.
Private Cloud
Private clouds are usually used by a single business or organization. It provides many of the benefits of public cloud, being scalibility, elasticity, and self-service, but requires the company's IT department to manage it. Private clouds also have the highest levels of reliability and security, although its services are limited to the private network.

Uses

Backup and File Storage

A primary use for cloud computing is for data storage. This becomes very useful when a person or company wants to secure their data in case of an emergency. Cloud storage prevents the need to build infrastructure or maintenance and instead relies on existing infrastructure in the form of server networks which actually make data recovery faster. The most prominent examples of cloud storage providers are Google Drive, Dropbox, and Amazon S3.

Test and Development

Cloud computing provides an environment to test and develop software. This allows for faster deployments while simultaneously saving on expenses. These environments can be easily scaled and are provided by companies like BlazeMeter and LoadStorm which simplify setting up and dismantling testing environments.

Big Data Analytics

Nowadays, companies need to analyze massive amounts of data to look for marketing techniques or patterns that can increase revenue. Cloud computing accomplishes this by providing massive amounts of computing power and more sophisticated resources. This includes tools like Hadoop, HPCC, or Cassandra that also make it easy to collect and analyze data.

Social Networking

Social networking is actually the most common use of cloud computing. Companies like Facebook, Twitter, Linkedin, etc, use the cloud to look for people with similar information and in doing so finds people you might now. This all requires powerful systems that are able to store and manage data effectively with cloud computing filling that niche.

Productivity

With options like Microsoft Office 365 or Google Suite, people can work on documents anytime, anywhere. These tools keep your data very secure since it’s being kept in the cloud where security is purely reliant on the provider’s security. Furthermore, these tools allow for people to work on the same documents at the same time, making collaboration very easy.
